# 🛡️ CyberShield AI

Учебный Python-проект по теме **искусственный интеллект и машинное обучение**: система определяет, похож ли сайт на фишинговый, по признакам URL и веб-страницы.

## Что внутри

- `generate_dataset.py` — создаёт синтетический датасет `data/phishing_sites_synthetic.csv`.
- `train_model.py` — обучает модель RandomForest и сохраняет её в `models/phishing_detector.joblib`.
- `app.py` — веб-интерфейс для проверки URL, анализа датасета и просмотра метрик.
- `data/` — готовый CSV-датасет.
- `models/` — обученная модель и метрики.

## Идея датасета

Датасет синтетический, но признаки подобраны реалистично:

- длина URL;
- количество точек, дефисов и цифр;
- наличие IP вместо домена;
- наличие `@` в адресе;
- HTTPS;
- подозрительные слова: `login`, `verify`, `bonus`, `secure`, `account` и т.д.;
- возраст домена;
- доля внешних ссылок;
- количество форм, pop-up окон и редиректов;
- валидность SSL-сертификата.

Целевая переменная:

- `0` — легитимный сайт;
- `1` — фишинг.

## Быстрый запуск

```bash
pip install -r requirements.txt
python generate_dataset.py
python train_model.py
streamlit run app.py
```

После запуска откройте ссылку, которую покажет Streamlit.

## Пример использования

В интерфейсе можно ввести URL, например:

```text
http://sber-login-secure.top/verify/account?session=932811
```

Затем указать дополнительные признаки страницы и получить вероятность фишинга.

## Модель

Используется `RandomForestClassifier` из `scikit-learn`.

В пайплайне:

1. `StandardScaler`;
2. `RandomForestClassifier`.

Метрики сохраняются в `models/metrics.json`:

- Accuracy;
- F1-score;
- ROC-AUC;
- Confusion Matrix.
